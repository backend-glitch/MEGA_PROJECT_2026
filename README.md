# 🚀 GSAP Animated Portfolio

A modern, interactive developer portfolio built using **HTML, CSS, JavaScript, and GSAP**.  
This project showcases multiple JavaScript projects with smooth scroll-based animations and section transitions.

---

## ✨ Features

- 🎬 Smooth scroll animations using GSAP + ScrollTrigger  
- 🧩 Multiple project sections with grouped layouts  
- ↔️ Alternating left/right transition effects  
- 📌 Pin-based animation sections  
- 🔗 Internal navigation between portfolio and project pages  
- ⚡ Page transition effects (fade in/out)  
- 📱 Responsive and clean UI  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3 (Flexbox + Grid)  
- JavaScript (ES6)  
- GSAP (GreenSock Animation Platform)  
- ScrollTrigger Plugin
